# 桌面文件计数技能

## 技能概述

此技能用于统计用户桌面目录下的所有文件数量，包括子目录中的所有文件。

## 使用场景

当用户请求以下操作时使用此技能：
- "统计桌面有多少个文件"
- "查看桌面文件数量"
- "计算桌面文件总数"
- "桌面有多少个文件"

## 使用方法

### 步骤 1：获取桌面路径

在 Windows 系统上，桌面路径通常是：
- `C:\Users\<用户名>\Desktop`

### 步骤 2：执行统计脚本

运行 `scripts/count_desktop_files.py` 脚本：

```
python scripts/count_desktop_files.py [桌面路径]
```

如果不提供参数，脚本会自动尝试获取默认桌面路径。

### 步骤 3：返回结果

将统计结果以友好格式展示给用户，包括：
- 文件总数
- 目录数量（可选）
- 按文件扩展名分类的统计（可选）
- 最大的文件（可选）

## 脚本说明

- `scripts/count_desktop_files.py`: Python 脚本，遍历桌面目录并统计文件数量

## 依赖

- Python 3.x（标准库，无需额外安装）
